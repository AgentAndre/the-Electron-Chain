5 · Figma Setup Annotations
Grid & Layout
  • • Frame: 1440 × 900 Desktop
  • • 12-column grid, 14px gutter, 20px margin
  • • Card padding: 14px (all sides)
  • • Card border-radius: 8px
  • • Card border: 1px #27272a
  • • Card background: #0f0f12
  • • Inner containers: #18181b (bg2)
  • • Auto Layout: Vertical, 12px gap
Interactivity (Prototype)
  • • WS Live Dot: pulse animation (opacity 1→0.35→1, 2s)
  • • Battery bar: width transition 300ms ease
  • • Price color: instant on data update
  • • Hover states: background → bg3 (#27272a)
  • • Map: Leaflet → no Figma equivalent (use placeholder)
  • • Chart bars: CSS height, 1px gap, flex align-end
  • • Mode badge: 4 states, color-coded
  • • Order rows: hover → bg3, cancel button fades in
PV Overlay (48h Chart)
  • • Amber semi-transparent fill: rgba(245,158,11,0.15)
  • • Top border: 1px rgba(245,158,11,0.4)
  • • Positioned absolute, behind price bar (z-index 0)
  • • Height normalized to same 150px scale as prices
  • • Shown only when pvForecast48h.length > 0
  • • Data source: PVSimulator (Spencer + isotropic sky)
Data Sources
  • • EPEX prices: HA sensor → Energycharts API → Demo
  • • PV forecast: PVSimulator (Spencer + isotropic sky)
  • • Node states: HA API every 1min, demo every 1min
  • • Order book: auto-refresh on WS events
  • • Blockchain: @peaq-network/sdk v0.2, demo fallback
  • • Map tiles: CartoDB Dark (cdn.jsdelivr.net/leaflet)
